# Procesos Judiciales

## Descripción

Este módulo permite la gestión completa de procesos judiciales en Odoo 18.

## Características

- Crear y gestionar procesos judiciales
- Seguimiento de estados (Borrador, Activo, Cerrado)
- Fechas de inicio y fin
- Integración con el sistema de mensajes y actividades

## Instalación

1. Descargar el módulo en el directorio de addons
2. Actualizar la lista de módulos
3. Buscar e instalar "Procesos Judiciales"

## Uso

1. Ir a Procesos Judiciales
2. Crear un nuevo registro con los datos del proceso
3. Gestionar el estado y las fechas según sea necesario

- base
